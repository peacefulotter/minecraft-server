# mc-server

## Installation

Install dependencies:

```bash
bun install
```

Then, create the required json arrays

```bash
bun run ./static/mapper.ts
```

## Run

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.0.22. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
